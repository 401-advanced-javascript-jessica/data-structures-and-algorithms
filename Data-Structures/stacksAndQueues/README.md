## Stack and Queue
Build Stack and Queue based on linked list

### Challenge
* The challenge was to build Stack and Queue and their relevant methods

### Approach & Efficiency
 #### Stack
 * push() - O(k)
 * pop() - O(k)
 * peek() - O(k)
 
 #### Queue
 * enqueue() - O(N)
 * dequeue() - O(k)
 * peek() - O(k)
